# Resource recommendations

The recommended resource requests for each Bigbytes component are as follows:

## bigbytes

- **CPU**: 475m
- **Memory**: 1.5Gi
- **Ephemeral Storage**: 1Gi

## browserlessChrome

- **CPU**: 1
- **Memory**: 1Gi
- **Ephemeral Storage**: 1Gi

## scheduler

- **CPU**: 475m
- **Memory**: 1425Mi
- **Ephemeral Storage**: 1Gi
