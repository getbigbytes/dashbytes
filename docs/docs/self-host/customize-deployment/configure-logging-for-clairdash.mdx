# Configure logging for Clairdash

By default Clairdash logs to the console in a human readable format. You can configure the logging behaviour, such
as logging to a file or using a JSON format, by
using the following environment variables:

| Variable                       | Description                                                 | Required? | Default                |
| ------------------------------ | ----------------------------------------------------------- | --------- | ---------------------- |
| `CLAIRDASH_LOG_LEVEL`          | The minimum level of log messages to display                |           | `INFO`                 |
| `CLAIRDASH_LOG_FORMAT`         | The format of log messages                                  |           | `pretty`               |
| `CLAIRDASH_LOG_OUTPUTS`        | The outputs to send log messages to                         |           | `console`              |
| `CLAIRDASH_LOG_CONSOLE_LEVEL`  | The minimum level of log messages to display on the console |           | `CLAIRDASH_LOG_LEVEL`  |
| `CLAIRDASH_LOG_CONSOLE_FORMAT` | The format of log messages on the console                   |           | `CLAIRDASH_LOG_FORMAT` |
| `CLAIRDASH_LOG_FILE_LEVEL`     | The minimum level of log messages to write to the log file  |           | `CLAIRDASH_LOG_LEVEL`  |
| `CLAIRDASH_LOG_FILE_FORMAT`    | The format of log messages in the log file                  |           | `CLAIRDASH_LOG_FORMAT` |
| `CLAIRDASH_LOG_FILE_PATH`      | The path to the log file                                    |           | `./logs/all.log`       |
